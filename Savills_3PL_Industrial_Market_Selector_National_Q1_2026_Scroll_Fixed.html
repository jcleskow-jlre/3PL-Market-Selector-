    Data sources and scoring methodology

    This version uses actual Savills Q1 2026 market statistics for rent, vacancy, inventory, under-construction pipeline, deliveries and YTD net absorption. The tool normalizes each variable onto a 0-100 scale, then lets the user weight the priorities. Infrastructure, labor and customer reach are transparent directional proxy scores that should be replaced with client-specific data before being used as a final site-selection model.

    VariableCurrent data statusScoring logic
