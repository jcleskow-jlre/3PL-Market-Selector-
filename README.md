# Savills 3PL Network Optimization Platform v4

Client-ready prototype for 3PL market ranking, transportation infrastructure, labor screen, real estate economics, renewal leverage and network strategy.

## Data layers reflected
- Savills Q1 2026 U.S. Industrial Market Research: rent, vacancy, inventory, under construction, deliveries and absorption.
- Public-data scoring framework for BLS OEWS labor, BLS QCEW logistics employment, BTS port TEUs, NTAD intermodal freight facilities, air-to-truck freight facilities and parcel-hub proximity.

## Notes
This is a standalone static HTML prototype. It can be deployed directly via GitHub Pages by uploading `index.html` to a repository and enabling Pages.

For final client-decision use, automate public dataset refreshes and load client-specific portfolio, customer, drive-time and lane data.
